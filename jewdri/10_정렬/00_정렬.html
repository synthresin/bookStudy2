버블정렬(bubble sort)

  • 정렬 알고리즘 중에서 가장 단순하기 때문에 언제나 1번 타자.
  • 단순한만큼 실행시간은 최악
  • 인접한 두 원소를 모두 다 비교하고 그 결과에 따라 두 원소의 위치를 서로 바꾼다.
  • 원소가 정렬돼가는 모습이 마치 수면 위로 떠오르는 거품(버플) 같다고 하여 버플정렬이란 이름이 붙었다.
ArrayList.prototype.bubbleSort = function(){
    var length = this.array.length;
    for(var i=0; i<length; i++){
        for(var j=0; j<length-1; j++){
            if(this.array[j] > this.array[j+1]){
                this._swap(j, j+1);
            }
        }
    }
}

  • 첫 번째 원소부터 마지막 원소까지 순회한다.(바깥쪽루프)
  • 안쪽루프는 첫번째 원소부터 끝에서 두번쨰 원소까지 순회하면서 현재원소와 그 다음 원소를 비교한다.
  • 이 두 원소가 정렬되어 있지 않다면, 위치를 서로 바꾼다.

교환 기능을 전담할 swap 함수를 작성 (ArrayList 내부에서만 사용가능한 프라이빗 함수)

ArrayList.prototype._swap = function(index1, index2){
    var aux = this.array[index1];
    this.array[index1] = this.array[index2];
    this.array[index2] = aux;
}

개선된 버블정렬

바깥쪽 루프를 두번 째 반복할때, 숫자 4,5는 이미 정렬이 끝난 상태다.
사실 굳이 비교할 필요가 없음에도 버블정렬은 일일이 모든 원소를 비교한다.
안쪽루프에서 바깥쪽 루프의 반복횟수를 차감하면 불필요한 비교작업이 준다.

선택정렬(selection sort)

selection sort는 제자리(in-place) 정렬 알고리즘의 하나로, 최솟값을 찾아 맨 앞으로 보내고, 그 다음으로 작은값을 찾아 2번째 위치로 보내는 식으로 정렬한다.
ArrayList.prototype.selectionSort = function(){
    this.sortTime = 0;
    var length = this.array.length;
    var indexMin;
    for(var i=0; i<length-1; i++){
        indexMin = i;
        for(var j=i; j<length; j++){
            if(this.array[indexMin] > this.array[j]){
                indexMin = j;
            }
            if(i !== indexMin){
                this._swap(i, indexMin);
            }
            this.sortTime++;
        }
    }
}

삽입정렬(insertion sort)

  • 한 번에 한 원소씩 정렬된 배열을 만들어가는 알고리즘
  • 첫번째 원소는 정렬이 끝났따고 가정하고, 두번째 원소와 비교해 첫번째 원소보다 더 작다면 첫번째 원소 앞으로 옮긴다.
  • 처음 두 원소의 정렬이 끝나면 다음엔 세번째 원소와의 비교를 계속한다.
ArrayList.prototype.insertionSort = function(){
    this.sortTime = 0;
    var length = this.array.length;
    var j, temp;
    for(var i=1; i<length; i++){
        j=i;
        temp = this.array[i];
        while(j>0 && this.array[j-1] > temp){
            this.array[j] = this.array[j-1];
            j--;
            this.sortTime++;
        }
        this.array[j] = temp;
    }
}

병합정렬(merge sort)

  • 실전에 응용된 정렬 알고리즘
  • 복잡도가 O(n log n)으로 훨씬 뛰어난 성능
  • 병합정렬 알고리즘의 핵심은 분할과 정복이다.
  • 정렬한 배열을 원소가 하나뿐인 배열 단위로 나뉠 때까지 분할하고, 반대로 이렇게 분할된 배열을 점점 더 큰 배열로 병합하면서 정렬을 완성한다.
  • 분할/정복이라는 접근 방식 덕분에 재귀호출 !!
