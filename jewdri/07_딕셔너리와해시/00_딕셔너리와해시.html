딕셔너리?

  • 딕셔너리(dictionary)는 데이터를 [키, 값] 쌍으로 담아두는 자료 구조
  • 키는 원소를 찾기 위한 식별자
  • 집합이 [키, 키], 딕셔너리가 [키, 값] 형태의 원소를 모아놓은 공간이라는 점에서 비슷
  • 딕셔러니는 맵(map)이라고도 한다.

딕셔너리 만들기

   this.items = {};
   this.length = 0;

  • set(키, 값) : 딕셔너리에 원소를 추가한다.
  • remove(키) : 키에 해당하는 원소를 삭제한다.
  • has(키) : 키에 해당하는 원소가 딕셔너리에 존재하면 true를, 그렇지 않으면 false를 반환한다.
  • get(키) : 키에 해당하는 원소의 값을 반환한다.
  • clear() : 모든 원소를 삭제한다.
  • size() : 원소의 개수를 반환한다. 배열의 length 프로퍼티와 비슷하다.
  • keys() : 딕셔너리의 모든 키를 배열로 반환한다.
  • values() : 딕셔너리의 모든 값을 배열로 반환한다.

has와 set 메소드

Dictionary.prototype.has = function(key){
    return key in this.items;
}
Dictionary.prototype.set = function(key, value){
    if(!this.has(value)){
        this.items[key] = value;
        this.length += 1;
        return true;
    }
        return false;
}

Dictionary.prototype.remove = function(key){
    if(this.has(key)){
        delete this.items[key];
        this.length -= 1;
        return true;
    }
    return false;

get과 values 메소드

Dictionary.prototype.get = function(key){
    return this.has(key) ? this.items[key] : undefined;
}
Dictionary.prototype.values = function(){
    var values = [];
    for(var k in this.items){
        if(this.has(k)){
            values.push(this.items[k]);
        }
    }
    return values;
}

집합연산

집합에서 사용 가능한 연산은 네가지가 있다.
  • 합집합 : 두 집합 중 어느 한쪽이라도 포함된 원소로 구성된 집합을 구한다.
  • 교집합 : 두 집합 모두 포함되어 있는 원소로 구성된 집합을 구한다.
  • 차집합 : 첫 번째 집합에는 있지만 두 번째 집합에는 없는 원소로 구성된 집합을 구한다.
  • 부분집합 : 어떤 집합이 다른 집합의 일부인지 확인한다.

합집합(Union)

  • 합집합은 수학기호로 A∪B와 같이 표시
  • A, B 어느 한쪽에 속하는 원소 x의 집합
Set.prototype.union = function(otherSet){
    var unionSet = new Set();
    var values =  this.values();
    for(var i=0; i<values.length; i++){
        unionSet.add(values[i]);
    }
    values =  otherSet.values();
     for(var i=0; i<values.length; i++){
        unionSet.add(values[i]);
    }
    return unionSet;
}

교집합(intersection)

  • 교집합은 수학기호로 A∩B와 같이 표시
  • A에도 속하고 B에도 속하는 원소 x의 집합
Set.prototype.intersection = function(otherSet){
    var intersectionSet = new Set();
    var values = this.values();
    for(var i=0; i<values.length; i++){
        if(otherSet.has(values[i])){
            intersectionSet.add(values[i]);
        }
    }
    return intersectionSet;
}

차집합(difference)

  • 차집합은 수학기호로 A-B와 같이 표시
  • A에는 속하지만 B에는 속하지 않는 원소 x의 집합
Set.prototype.difference = function(otherSet){
    var differenceSet = new Set();
    var values = this.values();
    for(var i=0; i<values.length; i++){
        if(!otherSet.has(values[i])){
            differenceSet.add(values[i]);
        }
    }
    return differenceSet;
}

부분집합(subset)

  • 부분집합은 수학기호로 A⊂B와 같이 표시
  • A의 모든 원소는 반드시 B에 존재해야 함을 의미
Set.prototype.subset = function(otherSet){
    if(this.size() > otherSet.size()){
        return false;
    }else{
        var values = this.values();
        for(var i=0; i<values.length; i++){
            if(!otherSet.has(values[i])){
                return false;
            }
        }
        return true;
    }
}
