### 목적

### 방식

- 매주 일요일마다 책을 기준으로 스터디를 진행
- 각 구성원당 깃의 브랜치를 이용해서 각자의 학습 내용 업로드
- 일요일에 모여서 각자의 학습 내용 공유 및 피드백
